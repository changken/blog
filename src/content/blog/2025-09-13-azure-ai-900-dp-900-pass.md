<p>繼我上次去考 AZ-900 大概過了1個月左右，我記得上次說我沒打算在 azure 平台上發展，所以 AZ-900 會是我唯一一張 Azure 認證。但由於我有收集癖好，想把 Azure 幾張基礎級別也收入囊中，所以在9/6 號我又去考了兩張 AI-900 與 DP-900 的基礎級別的 Azure 考試了，順便也把自己打臉了。</p>



<p>先預告，下一張 Microsoft 基礎級別考試就是 SC-900，所以後續等考完再說。</p>



<p>再預告下一篇，我特地請 9/11下午的假去恆毅考 AWS SAA，所以後續會有一篇較長篇的 SAA 心得。</p>



<!--more-->



<h2 class="wp-block-heading">線上教學資源:</h2>



<p>老樣子，微軟爸爸的 virtual training date 都有2~4小時的線上教學，你可以任選時間進去聽課，然後完整參與可以獲取 50% discount coupon，可以讓該次考試變為25美金。</p>



<p>microsoft learn 也有兩個認證的學習資料+練習評量，我的建議是也順便看一遍，至少在看教學影片時會比較有感，不容易看到看不下去。</p>



<p>除了官方資源外，我也推薦採用 freecodecamp 上 exampro 出品的兩部教學。</p>

<h2 class="wp-block-heading">正式考試</h2>



<p>一樣老地方，我選南京復興的精誠恆毅中心的考場。</p>



<p>我先進去考 AI-900，大概 20 分鐘交卷。再接著進去考 DP-900，大概 20~30 分鐘交卷。</p>



<p>後續到櫃台拿包包，成績考完當下就會顯示並給予紙本成績單。</p>



<h2 class="wp-block-heading">成績 </h2>



<p>AI-900: 873分</p>



<p>DP-900: 880分</p>



<h2 class="wp-block-heading">心得</h2>



<p>考試內容基本上不會到太難，如果有刷題到的話就會遇到 1~2 題考古題，所以一定要記得刷題唷。我個人覺得 Hands On 可以考慮做一下，稍微留個肌肉記憶會比較好做題目。</p>



<p>考試基本上不難，如果你是軟體工程師 + 有寫過機器學習領域的論文，等於又得到一次不錯的複習機會，又可以得到不錯的標章。</p>

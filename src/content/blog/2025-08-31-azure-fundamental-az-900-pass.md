<p>AZ-900 是一個 Azure 服務的基礎考試，個人覺得比 AWS CCP 簡單。 因此我也打算順便把 AZ-900也一起考掉，以便後續可以專心主攻 AWS 雲。</p>



<!--more-->



<h2 class="wp-block-heading">閱讀文件教學與影片<strong>準備</strong></h2>



<p>有關於考試準備的部分，大家可以上「Microsoft Learn」把 AZ-900 先跑一遍。官方教學很清楚，有附贈 Sandbox 環境，一日內可以開啟10次沙盒。所以倒是不需要註冊200美金1個月的試用帳號。</p>



<pre class="wp-block-preformatted">https://learn.microsoft.com/zh-tw/credentials/certifications/azure-fundamentals/?practice-assessment-type=certification</pre>



<p>有關於線上影片資源的部分，推薦直接報名官方的「Virtual Training Day」</p>



<p>該活動在活動前可以先註冊報名，可以選 subtitle 有繁體中文的場次，不過授課內容都是英文授課。</p>



<p>我當初是選 english subtitle，所以當作順便練習英文聽力。</p>



<p>上完後隔一天會給予 50% 的 Exam discount coupon，Fundamental 等級的考試是 50 USD，打半價就是可以減免考試費用 25 USD，剩下 25 USD 要刷卡繳費。</p>



<p>後續的 AI-900, DP-900 也是一樣的道理，我都有上完，但評估後續不會在 Azure 發展所以先不考 。</p>



<pre class="wp-block-preformatted">https://www.microsoft.com/en-us/events/category/microsoft-virtual-training-days?filters=delivery-language%3Aenglish&amp;scenario=mvtd</pre>

<h2 class="wp-block-heading">實際至考試中心考試</h2>



<p>因此這次我是使用 這次我又到恆毅訓練中心來考 AZ-900 的考試，他也是使用 pearson vue 考試平台來做考試，只是比較不方便的是每次從 microsoft learn 跑到考試平台都要完整填寫一次個人資料。</p>



<p>考試可以選擇英文、繁體中文、簡體中文、日文等，選繁體中文也可以看到原始英文題目，所以不用擔心可以直接選繁中考試，但請不要完全依賴題目的翻譯，因為大部分的翻譯品質真的有待加強!!!</p>



<p>這次我比較勇敢我刷了差不多的前一天去預約考試，結果到了當天，跟 AWS CCP 一樣，恆毅中心都會嚴格檢查護照、中文證件、背包、眼鏡、褲子口袋等。考試時會給一張畫畫版+畫筆可以讓你做筆記。</p>



<p>考試完會馬上知道分數+通過與否，到櫃台拿包包時會有一份紙本成績單供你帶回參考。上午考完, 下午就會有認證通過的 email + 電子證書，可以把它加到 linkedin 與 credly 上面，蒐集標章。</p>



<h2 class="wp-block-heading">考後心得</h2>



<p>基本上題目涵蓋 Compute, Storage, Network, Database 四大服務。 </p>



<p>SAAS, PAAS, IAAS 必考,  一些像是 Entra ID, SubScription, Mangement Group,  Resource Group, Resource Lock(資源鎖定), Principal(原則), 安全 + 和規，有很多題目都是從這些地方延伸過來的。</p>



<p>題目形式: 選擇題, 是非題, 配對題, 拖拉排序選項題, 給你實際UI，要你點選哪個圖示代表甚麼服務的選擇題</p>



<p>官方模擬考題會比實際考試簡單一些，但其餘第三方的模擬考試幾乎都是比較難的。</p>



<p>除了az900practices 那個感覺難到 az-104, az-204 的等級，幾乎我都做錯外。</p>



<h2 class="wp-block-heading">結論</h2>



<p>AZ-900 是一場 Azure 服務的基礎考試，基本上有考過 AWS CCP 會變得輕鬆很多，我是很推薦 AWS CCP 的持有者也順便把他考掉，這樣還有一些 Azure 服務的基礎知識，還不至於會只局限於 AWS 環境中。</p>



<p>實際上有沒有用途嘛，我個人體感是沒有任何卵用，那些 Job Hunter, HR 大多都只看 associate professional 等級的證書，但如果未來要往上考的話是一個很不錯的階段性認證。</p>

<p>在上個月（10/25），我又去考了一張 aws ai 證照。這是因為 aws 又有 AI 認證的活動可以參加，並且報名學習後就會收到一張 voucher，所以我就去順手拿了一張 50% off 的 voucher 了。</p>



<!--more-->



<p>課程部分我是使用 stephane maareak 的教材，模擬考則是去 之前 aws 小編有給 question bank，以及 stephane 課程後面隨附的 20 題模擬考。我個人都是題海戰術派的，能夠刷題就往死裡刷，因此我覺得題目不夠多，後續我再去 youtube 找有標注aif c01 的模擬考考題來做使用。</p>



<p>考完後，我覺得實際考試難度會比 stephane 的低，差不多就是 question bank以及網路上的難度。stephane 老師的題目真的覺得偏難，參考就好。</p>



<p>至於課程部分，當然你也可以使用 freecodecamp Exampro Andrew Brown 的教材，在 youtube 上面都有影片連結，並且「無料」。</p>



<p>如果你是有上過台大李宏毅老師的機器學習、生成式 ai 等課程的話，實際上過這些業師的課程，我個人是覺得，他們講得會比李宏毅老師的還要淺，如果你本身是有上過 ai 相關的課程，這應該是可以輕鬆考取的一張證照。</p>



<p>業師講的內容與李宏毅老師講內容，兩者差別在於 aws ai 課程要多學 aws 提供的 ai 服務，舉例像是 aws sagemarker, aws bedrock 等平台，AWS Transcribe、Textract等managed service，其餘的機器學習基本知識基本上都是通用的，不需要重學。又或者你是可以先上李宏毅的，再補業師的 aws ai 服務內容。</p>

<h2 class="wp-block-heading">成績</h2>



<p>分數：828 通過</p>



<p>考試地點：恆毅教育訓練中心（老地方）</p>



<p>考試時間：9：30~11：30</p>



<p>在官網查詢到成績的時間：19：00</p>



<p>收到 credlly 標章時間：19：22</p>



<p>收到考試祝賀的email時間：隔天 04：26 左右</p>



<p>請注意！只有 aws ccp 才會馬上給成績，其餘的考試都是要等 6~8 小時，甚至可以到 24~48 小時才會知道結果。</p>



<p>官方標注的 SLA 是 5 個工作天。</p>

<p>大家好，上次有跟大家預告說我會分享SC-900的考試經驗。所以我就花了3~7天的時間把 microsoft learn  給跑一遍，並且也去報名「Microsoft 安全 稽核 合規」的虛擬培訓日的活動，只是該活動需要等到 9/29, 9/30 才會開始播送教學。講白了就是我不想浪費時間在 SC-900 的學習與刷題上。所以我就自己先去 youtube 找教學影片，並且也去 youtube 上找模擬試題。</p>



<!--more-->



<p>我個人體感難度 AZ-900 &gt;&gt;&gt; SC-900 &gt;&gt; DP-900 &gt;= AI-900。</p>



<p>我個人覺得 AZ-900 考得算滿廣的，所以真的花了2~2.5周的時間讀 microsoft learn 並且也去上虛擬培訓日，並且有提到很常常考 EntraID + Management Group,  Subscription, Resource Group, Resource 層級的題目。相比之下 SC-900 大概 5~7 天可以搞定，不過在讀的過程實在是會很矇逼 + 頭腦爆炸，因為會有一堆名詞在腦袋中打轉。並且我的 Azure Student 帳號由於權限不夠，所以進不去 EntraID，進得去 Purview, Priva, Sentiel, Intune, MicroSoft Denfender 等 dashboard，但有些功能就是會被限制，所以變得我沒辦法跟著進去 dashboard 試試看，自然就會很難以想像跟學習。</p>



<p>這邊我比較推薦直接死記硬背吧！</p>

<h2 class="wp-block-heading">心得</h2>



<p>總分數: 861，20~25分鐘提前交卷。AI-900, DP-900 也是一樣的時間提前交。</p>



<p>我的體感是 Azure 安全相關大概佔 10~15%，其餘的就是偏向 EntraID(舊Azure Active Directory), 安全(defender), 合規(purview) 之類的。</p>



<p>如果是為了 Azure 而考的，可能會略為失望，但如果是為了後面三個項目而考的，並且下一張確定要往 SC-200, SC-300, SC-400 走的話，是一個不錯的 fundamental 級別的考試。</p>
